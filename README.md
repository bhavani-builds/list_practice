# 📘 Module 3: Data Structures in Python - Lists

## 📌 Overview

This repository contains solutions to **Module 3: Data Structures (Lists)** assignments implemented in **Python**. These exercises cover the fundamental operations on Python lists, including list creation, slicing, comprehensions, sorting, nested lists, matrix operations, list manipulation, and more.

These assignments are ideal for beginners learning Python and preparing for coding interviews.

---

## 🚀 Topics Covered

* Creating Lists
* Accessing List Elements
* List Slicing
* List Comprehensions
* Filtering Lists
* List Methods
* Nested Lists
* List of Dictionaries
* Matrix Transposition
* Flattening Nested Lists
* List Manipulation
* List Zipping
* List Reversal
* List Rotation
* List Intersection

---

## 📂 Assignments

### ✅ Assignment 1: Creating and Accessing Lists

* Create a list containing the first 20 positive integers.
* Print the list.

---

### ✅ Assignment 2: Accessing List Elements

* Print the first element.
* Print the middle element.
* Print the last element.

---

### ✅ Assignment 3: List Slicing

* Display the first five elements.
* Display the last five elements.
* Display elements from index 5 to index 15.

---

### ✅ Assignment 4: List Comprehensions

* Generate a list containing the squares of the first 10 positive integers using list comprehension.

---

### ✅ Assignment 5: Filtering Lists

* Extract only the even numbers from a list using list comprehension.

---

### ✅ Assignment 6: List Methods

* Generate random numbers.
* Sort in ascending order.
* Sort in descending order.
* Remove duplicate values.

---

### ✅ Assignment 7: Nested Lists

* Create a 3×3 matrix.
* Print the matrix.
* Access an element using row and column indices.

---

### ✅ Assignment 8: List of Dictionaries

* Create student records using dictionaries.
* Sort students by score in descending order using a lambda function.

---

### ✅ Assignment 9: Matrix Transposition

* Write a function to transpose a 3×3 matrix.
* Display both the original and transposed matrices.

---

### ✅ Assignment 10: Flattening a Nested List

* Convert a nested list into a single list using list comprehension.

---

### ✅ Assignment 11: List Manipulation

* Remove elements at specified indices.
* Insert a new element at a given position.

---

### ✅ Assignment 12: List Zipping

* Combine two lists into a list of tuples using the `zip()` function.

---

### ✅ Assignment 13: List Reversal

* Reverse a list using slicing.

---

### ✅ Assignment 14: List Rotation

* Rotate a list by a specified number of positions using slicing.

---

### ✅ Assignment 15: List Intersection

* Find the common elements between two lists using list comprehension.
